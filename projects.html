May 2022 - July 2022
Land Use Land Cover Classification using SAR Images and Deep Learning Technique
  • Project focuses on using Sentinel-1 SAR images for land use and land cover classification in Ahmedabad and Gandhinagar.
  • Data acquisition involves downloading Sentinel-1 SAR images and performing pre-processing using the SNAP Sentinel 1 Toolbox.
  • Terrain correction is applied using a digital elevation model (DEM) to correct geometric distortions.
  • Supervised classification utilizes the random forest classifier with training areas selected in SNAP.
  • UNet model, a convolutional neural network architecture, is trained for semantic segmentation of satellite images.
  • Results show 85% accuracy across ten land cover classes, demonstrating the potential for globally consistent LULC maps at 5m resolution.
  • Sentinel-1 SAR image
  • SNAP Toolbox
  • UNet model
  • Google Colab
